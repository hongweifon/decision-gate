## Contributing to Engineering Re-planning

### What We're Looking For

- **New trigger signals**: Real-world patterns where the agent kept going when it shouldn't
- **Threshold data**: Evidence for tuning thresholds (false positive / false negative rates)
- **Edge case reports**: Scenarios where the skill misbehaved

### Process

1. Open an issue first for anything beyond a typo fix
2. Describe the problem, not the solution
3. Include a concrete example session transcript if possible

### Development

Edit SKILL.md. Test by installing locally and running a long task.

### Code of Conduct

Be direct, be constructive, be technical.
