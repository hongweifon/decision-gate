# Example: Research
---
Task: Research a topic and compile findings for a report.
---
Execution: Agent searches, finds 5 authoritative sources. Sources
disagree on key facts. Each source cites different data. Scope expands:
verify each source, cross-check citations, find original data.
---
Health assessment: Requirement Health flagged. Manual judgment required
for source evaluation. Decision Health flagged: multiple valid approaches
to presenting conflicting information.
---
Policy check: No user waiver active.
---
Gate route: Coordination, Ask User.
Options presented:
A. Include all sources with comparative analysis: thorough, longer.
B. Prioritize most recent sources only: faster, may miss context.
C. Use most cited source as primary, note conflicts: balanced, medium effort.
---
Result without Decision Gate: Agent spends excessive time trying to
resolve ambiguity that requires human domain judgment.
Result with Decision Gate: User provides source preference. Agent
proceeds efficiently within chosen scope.
